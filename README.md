> Проверить эквивалентность двух заданных конечных автоматов-распознавателей с 4 и 5 состояниями.\
> ***Задание 2*** \
> Минимизировать заданный конечный автомат-распознаватель с 8 состояниями. \

> ***Задание 4*** \
